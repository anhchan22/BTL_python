## 🔧 Implementation Reference

### To Create a New Page with Neumorphism
**Follow pattern from:** `phase-02-authentication-refactor.md`
1. Use AuthCard wrapper
2. Use FormField for inputs
3. Use NeuButton for buttons
4. Apply design tokens (colors, shadows, radius)
5. Add focus states (ring-2 ring-neu-accent)
6. Test on mobile

### To Create a New Component
**Pattern:** See `phase2-quick-reference.md`
- Use Tailwind classes
- Reference CSS variables
- Keep components small and focused
- Export with clear prop documentation

### To Style an Element
**Hierarchy:**
1. Use pre-built utilities: `shadow-neu-extruded`
2. Use design token classes: `bg-neu-bg`, `text-neu-fg`
3. Use CSS variables directly: `color: var(--color-foreground)`
4. NEVER hardcode colors or shadows
