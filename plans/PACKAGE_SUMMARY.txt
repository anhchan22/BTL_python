## ✨ Features at a Glance

### 1️⃣ Notification Badge
```
User sees:  👤 (user button with badge)
            └─ 🔴 "3" (red notification badge)

How it works:
- Fetches unread count every 30 seconds
- Shows badge only if count > 0
- Animates with "pop in" effect
- Hides automatically at count = 0
```

### 2️⃣ Image Upload
```
User sees:  📸 Upload area
            ├─ Click or drag & drop
            ├─ Selected Images: 3/6
            └─ [Thumbnails with remove buttons]

How it works:
- Select 1-6 images (jpg, png, webp)
- Validate type, size, count
- Show live preview grid
- Remove individual images
- Submit with FormData multipart
```

### 3️⃣ Image Gallery
```
User sees:  ◀  [Main Image 16:9]  ▶
            └─ Thumbnails below
            └─ Counter: "2 / 5"

How it works:
- Display images from API response
- Navigate with buttons or thumbnails
- Show placeholder if no images
- Responsive on mobile
- Optional keyboard navigation
```
